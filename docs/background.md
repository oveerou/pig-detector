# 项目背景

## 行业背景

生猪养殖是农业产业的重要组成部分。随着规模化养殖的发展，猪只盘点、健康监测、行为分析等任务对人工依赖度高，效率低且容易出错。基于计算机视觉的智能化方案可以辅助养殖人员完成这些任务，降低人工成本并提升管理效率。

## 技术方向

本项目选择“可见光图像猪只检测与计数”作为切入点，这是智慧养殖中最容易落地、数据最容易获取的方向之一。通过目标检测模型识别图像中的猪只位置，并基于检测结果进行数量统计和密度分析。

## 论文参考

- 《生猪智能检测技术研究进展与未来展望》
- DOI: 10.12133/j.smartag.SA202507048

该论文为综述性文献，梳理了生猪智能检测技术的发展脉络。本项目参考其“可见光图像猪只检测 / 盘点 / 异常识别特征提取”分支，**不是对该论文的复现**。

## 岗位匹配

本项目针对牧原食品“算法工程师（提前批）”岗位，重点展示以下能力：

- 深度学习模型训练与调优
- 目标检测任务全流程实现
- 数据清洗、转换与审计
- 算法结果可视化与交互演示
- Python 工程化与代码组织能力
